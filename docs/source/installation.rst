Installation
============
The most recent release can be installed from
`PyPI <https://pypi.org/project/resp_gen_ai>`_ with:

.. code-block:: shell

    pip install resp_gen_ai

The most recent code and data can be installed directly from GitHub with:

.. code-block:: shell

    pip install git+https://github.com/chenxshuo/resp_gen_ai.git

To install in development mode, use the following:

.. code-block:: shell

    git clone git+https://github.com/chenxshuo/resp_gen_ai.git
    cd resp_gen_ai
    pip install -e .
